Paragraphs

Task 1 Selector specificity is an algorithm used by browsers to determine the CSS declaration that is most relevant to an element. It determines which rule from competing CSS declarations gets applied to an element. Helps browsers decide which style to apply when multiple rules target the same element. Example: If two rules conflict, the one with higher specificity wins.

Task 2 Complex selectors are sequences of one or more simple and/or compound selectors separated by combinators (e.g., descendant combinator, child combinator). They allow you to select elements based on their relationship to other elements within the document tree.

Task 3 Pseudo-classes are like special states or conditions that you can apply to an HTML element. They come into play when an element is in a specific state or being interacted with (e.g., hovered, clicked, focused). Pseudo-elements allow you to style specific parts of an element. They are created using double colons (`::`) in CSS. In general, pseudo-classes deal with states and interactions, while pseudo-elements target specific portions of an element.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
