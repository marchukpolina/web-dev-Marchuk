Safari

  1. Open Safari and click the "Safari" button in the menu bar.
  2. Select "Preferences" (or use the keyboard shortcut Command+)
  3. Go to the "Advanced" tab
  4. Check the box for "Show Develop Menu in Menu Bar"

Relative links are generally preferable for internal links in an HTML page. They make your code more portable and easier to maintain when moving content around within the same website or directory.

task 3

Absolute Links

  1. When linking to a different website.
  2. When linking to a resource not in the same directory as the current page.
  3. To ensure the link works even if the website changes its domain or protocol.

Relative Links

  1. When linking to a resource on the same website
  2. When linking to a resource in the same directory or a subdirectory of the current page.
  3. To make code more portable and easier to maintain.

task 4

JPEG (Joint Photographic Expert Group)

Good for lossy compression of still images (commonly used)

PNG (Portable Network Graphics)

Preferred over JPEG for more precise reproduction of source images or when transparency is needed.

GIF (Graphics Interchange Format)

Suitable for simple images and animations.

WebP

Excellent choice for both images and animated images, offering better compression than PNG or JPEG.

AVIF (AV1 Image File Format)

High-performance format for both images and animations, with better compression than PNG or JPEG

SVG (Scalable Vector Graphics)

Ideal for user interface elements, icons, and diagrams that need to be drawn accurately at different sizes.

task 5

WEBP Image Type

  • A format developed by Google
  • Offers better compression than PNG or JPEG
  • Supports higher color depths, animated frames, and transparency
  • Excellent choice for both images and animated images

task 6

SVG (Scalable Vector Graphics)

  • A vector image format
  • Ideal for user interface elements, icons, and diagrams.
  • Ensures accurate rendering at different sizes.
  • Supported by major browsers
